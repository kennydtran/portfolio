Flappy Plane

Project Overview

Flappy Plane is a side-scrolling game where players control a plane navigating through obstacles by tapping the screen, inspired by the classic Flappy Bird game. With improved graphics, dynamic obstacle generation, and customizable aircraft skins, this game offers an addictive and challenging experience for casual gamers.

Features

Challenges

The biggest challenge in developing Flappy Plane was balancing the difficulty level to make the game challenging yet enjoyable. Creating a smooth physics system for the plane's movement required multiple iterations to get right, while ensuring consistent performance across different devices presented additional technical hurdles.
