Crossy Roads

Project Overview

Crossy Roads is a 3D voxel-style arcade game where players navigate a character across roads, train tracks, and rivers while avoiding various obstacles. Inspired by the classic "Frogger" game, this modern reimagining offers endless levels, multiple playable characters, and vibrant, blocky environments.

Features

Challenges

Creating Crossy Roads posed significant technical challenges, particularly in implementing the procedural generation of endless levels while maintaining smooth performance. Optimizing the game for mobile devices required careful attention to polygon counts and texture sizes while preserving the charming voxel aesthetic.
